第三章：系统机制-映像加载器

发表于 2018-04-26 | 分类于 windows系统

映像加载器

         当系统中的进程被启动的时候，内核创建一个内核对象表示该进程，并执行各种和内核相关的初始化任务。绝大部分的初始化任务是在内核之外完成的，这些工作是由映像加载器完成的。映像加载器驻留在用户模式下的Ntdll.dll中，映像加载器完成以下的初始化工作：

  • 1.初始化其他用户模式，包括堆栈的初始化，TLS和FLS的初始化
  • 2.解析执行文件的IAT，DLL的导出表
  • 3.加载卸载DLL，维护模块数据库
  • 4.启用API集和API重定向

Linux常见命令-文件相关

发表于 2018-04-14 | 分类于 linux

第一部分：文件操作

  • 查看文件 ls [选项]

    • -a:查看隐藏文件
    • -l:查看详细信息
    • -d:查看指定路径
    • -h:查看文件大小（非字节显示）[要和-l参数合用]
    • -i:查看文件id

  • 创建目录 mkdir [目录名]

    • -p:递归创建

  • 切换目录 cd [目录]

    • .. ：回到上级目录
  • 删除目录 rmdir [目录]
    • -p： 递归删除空目录

  • 查看当前目录 pwd

  • 复制文件 cp [源文件/目录] [目标文件/目录]

    • -r:复制目录
    • -p:保存被复制文件的属性
    • 可以改名，在目标文件中修改

  • 剪切或者改名 mv [源文件/目录] [目标文件/目录]

    • 可以改名

  • 删除文件 rm [文件名称]

    • -r 删除目录
    • -f 强制执行

  • 创建文件 touch [文件]

    • 创建带有空格的文件，可以使用“”把文件名引起来

  • 查看文件内容 cat [文件名]

    • -n :显示行数

  • 查看文件内容 wc [] [文件名]

    • -l ：显示行数
    • -m ：显示字符数
    • -c : 显示字节数
    • -L ：显示最长行的长度

  • 反向查看文件 tac [文件名]

  • 分页查看文件 more [文件名]

    • 空格/f 翻页
    • Enter 换行
    • Q/q 退出

  • 可向上翻页查看文件 less [文件名]

    • PageUp 向上翻页
    • 上方向键 向上换行
    • / 搜索
    • n 搜索下一个

  • 查看文件前几行 head [文件]

    • -n 规定行数（默认是10行）

  • 查看文件末尾几行 tail [文件]

    • -n 规定行数
    • -f 实时显示[查看日志文件]

  • 创建文件链接 **ln [选项] [链接目录文件] [链接名]

    • -s:创建的是软连接（快捷方式）
    • 默认：硬链接（读写操作和源文件一样）

第二部分 文件权限

  • chmod 修改文件或者目录权限

    • chmod [{ugoa}{+-=}{rwx}] [目录]
    • -R 递归修改
    • chmod [{4(r)2(w)1(x)}] [目录/文件]
    • 普通用户即使没有w权限也可以删除root用户创建的文件（对文件的删除的必要条件是对文件所处在目录具有写入权限）

  • chown 修改文件所有者

    • chown [用户名] [文件/目录]

    • useradd [用户名] 创建用户

    • chgrp [组名] [文件/目录]

  • umask 显示默认权限

    • 显示的数值是权限的掩码，真实权限是777-该数值，例如数值为022，权限则为755
    • -S ：显示具体的权限（rwx）
    • umask [权限掩码] ：修改默认的权限数值

第三部分 文件查询

  • find [范围] [搜索条件] 搜索文件

    • -name
      • -name “init” :文件名中存在init
      • -name “inti??” :文件名前四个为init，后面两个任意
      • -iname：不分大小写
    • -size 根据文件大小
      • +n 大于n
      • -n 小于n
      • n 等于n
    • -cmin -时间 根据修改时间
    • -amin -时间 根据访问时间
    • -mmin -时间 根据文件内容
    • -a ： 表示两个条件都符合
    • -o ： 表示多个条件只需要符合一个
    • -type 按照文件类型查找
      • -f 文件
      • -d 目录
      • -l 链接
    • -inum 根据i节点来查找
    • -exec/-ok {}\; 调用其他命令。

  • locate [目录/文件] 搜索（需要更新文件数据库）

    • -i：不区分大小写
    • updatedb ：更新数据库

  • which [文件/目录] ：用于查找系统命令

    • 查看命令是否带有别名
  • grep [关键词] [搜索路径] ：搜索文件内容
    • -i 不区分不大小写
    • -v 排除指定字符
  • whereis [命令]
    • 查看命令的所在位置，查看配置文件的位置，和帮助文件的文职。一般的文件名是1的是命令的帮助文件，文件名是5的是配置文件的帮助

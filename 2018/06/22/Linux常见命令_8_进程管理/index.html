Linux常见命令<8>进程管理

发表于 2018-06-22 | 分类于 linux

进程管理

查看进程

  • ps：查看进程信息
    • -le：查看系统内所有进程的详细信息(-l:显示详细信息，-e:显示所有进程)
    • aux:查看系统内所有进程的详细信息
    • 显示的内容介绍

  • top：查看系统信息
    • -d：指定刷新时间间隔
    • P：以CPU使用率排序(默认)
    • M：内存使用率
    • N：以PID排序



  • pstree：查看进程树
    • -p：罗列进程树

      杀死进程

  • kill [信号] [PID]：杀死指定进程
    • 重启进程-1
    • 正常结束-15
    • 强制结束-9
  • killall [选项] [信号] [PID]：按照进程名杀死进程

  • pkill [选项] [信号] [进程名]：按照进程名杀死进程
    • -t 终端号：按照终端号剔除用户
  • 实例1：
    • w：查看登录用户
    • pkill -t [终端号]:剔除用户

      工作管理

  • jobs:查看后台命令
  • fg %工作号：将后台执行的命令恢复到前台

  • bg %工作号：将后台执行的命令恢复到后台

    系统资源查看

  • vmstat[间隔] [次数] ：查看系统资源
  • dmesg：查看开机内核信息(查看软硬件信息)
  • free：查看内存使用状态
    • k
    • m
    • g
  • uptime：看时间
  • uname：查看系统内核信息
    • -a：所有信息
    • -r：内核版本
    • -s：内核名称
  • file /bin/ls:判断操作系统位数(间接)
  • lsb_release -a:查看系统发行版本
  • lsof [选项]：列出进程使用或者打开的文件
    • -c [字符串]：
    • -u [uid]:
    • -p [pid]:

      系统定时任务

  • crontab

Linux常见命令(2)常用命令

发表于 2018-04-28 | 分类于 linux

第一部分 帮助

  • man [命令] /info
    • / [查询的内容] 查询指定的字符串
    • man [配置文件名称] ：显示配置文件的作用。如果直接加绝对路径，显示的是配置文件的内容
    • 有文件的帮助信息，也有配置文件的帮助信息。如果是man 5 [命令] 查看的是配置文件的帮助信息，如果是man 1 [命令] 显示的是命令的帮助信息【见linux(1) whereis命令】
  • whatis [命令] ：显示命令作用（简略）//whereis [命令]
  • apropos [配置文件] ：显示配置文件的作用
  • 命令 -h(–help)
  • help [命令] shell内置命令的帮助信息（shell编程中的命令）
    • 也可以查看命令的路径

      第二部分 用户管理

  • useradd [用户名] 添加用户
  • passwd [用户名] 为用户设置密码.
  • who 查看当前用户名(tty 本地登录 pts 远程登录)
  • w 登录用户和登录时间等具体信息
  • usermod [选项] [用户ID]
    • -g<群组>：修改用户所属的群组；
    • -G<群组>；修改用户所属的附加群组；
    • -l<帐号名称>：修改用户帐号名称；
    • -L：锁定用户密码，使密码无效；
    • -u：修改用户ID；

第三部分 压缩解压

  • gzip [文件名] ：压缩文件(只能压缩文件)，并且不保存源文件
    • gunzip /(gzip -d) [压缩文件]：解压缩
  • tar [选项] [压缩文件名] [目录] 针对目录
    • -c 打包 //zcf（选项有顺序关系）
    • -v 显示详细信息
    • -f 指定文件名
    • -z 打包同时压缩
    • -cjf 生成 tar.bz2文件
  • tar
    • -x 解包 [压缩文件]
    • -z 解压缩
    • 其他和压缩一样
    • -xjf 解压tar.bz2压缩包
  • zip [选项] [压缩文件] [带压缩文件]
    • -r 压缩目录
    • unzip [压缩文件] 解压缩
  • bzip2 [选项] [带压缩文件]
    • -k 保存源文件
    • 压缩比大
  • bunzip2 [选项] [压缩文件]
    • -k 是否保存压缩包

第四部分 网络命令

  • ping 测试网络通畅性
    • -c 数量 ：ping的次数
  • ifconfig
    • [网卡名称] [IP] ：网络配置信息
  • last 用户登录信息
  • lastlog
    • -u [用户ID]
  • traceroute [主机名] 追踪网络访问跃点

  • netstat 查询网络状态

    • -t ：查询TCP协议状态
    • -u : UDP协议状态
    • -l ：监听
    • -r ：路由(网关)
    • -n ：显示IP地址及端口
    • -a : 所有选项
    • 常用实例：
      • netstat -an: 查看本机所有的网络连接
      • netstat -tlun:查看本机监听的端口
      • netstat -rn:查看本机路由列表

  • mount [设备文件名] [挂载点]挂载

    • 1.创建一个挂载目录（/mnt下）
    • 2.mount
    • 3.umount [挂载点/设备文件名]卸载挂载点，但是不能在挂载点下卸载

第五部分 关机与重启

  • shotdown [选项] [参数]
    • -h：将系统关机
    • -r：系统重新启动
    • -c：中断关机操作
    • -t：设置执行时间(s)
    • xx:xx(表示时间):shutdown -h xx:xx 表示xx：xx时关机
    • now：参数表示立刻执行
  • reboot ：立刻重启

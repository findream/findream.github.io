Linux常见命令(1)文件相关

发表于 2018-04-14 | 分类于 密码学

  • 查看文件 ls [选项]

    • -a:查看隐藏文件
    • -l:查看详细信息
    • -d:查看指定路径
    • -h:查看文件大小（非字节显示）
    • -i:查看文件id

  • 创建目录 mkdir [目录名]

    • -p:递归创建

  • 切换目录 cd [目录]

    • .. ：回到上级目录

  • 删除目录 rmdir [目录]

  • 查看当前目录 pwd

  • 复制文件 cp [源文件/目录] [目标文件/目录]

    • -r:复制目录
    • -p:保存被复制文件的属性
    • 可以改名，在目标文件中修改

  • 剪切或者改名 mv [源文件/目录] [目标文件/目录]

    • 可以改名

  • 删除文件 rm [文件名称]

    • -r 删除目录
    • -f 强制执行

  • 创建文件 touch [文件]

    • 创建带有空格的文件，可以使用“”把文件名引起来

  • 查看文件 cat [文件名]

    • -n :显示行数

  • 反向查看文件 tac [文件名]

  • 分页查看文件 more [文件名]

    • 空格/f 翻页
    • Enter 换行
    • Q/q 退出

  • 可向上翻页查看文件 less [文件名]

    • PageUp 向上翻页
    • 上方向键 向上换行
    • / 搜索
    • n 搜索下一个

  • 查看文件前几行 head [文件]

    • -n 规定行数（默认是10行）

  • 查看文件末尾几行 tail [文件]

    • -n 规定行数
    • -f 实时显示[查看日志文件]
